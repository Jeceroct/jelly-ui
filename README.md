# JellyUI

## 概述
JellyUI 是一个基于 Vue 3 和 TypeScript 的 UI 组件库，提供了丰富的组件和工具，帮助开发者快速构建用户界面。
JellyUI 的设计语言是柔和、年轻、易用的，注重用户体验和交互效果。

## 特性
- 基于 Vue 3
- 提供 TypeScript 智能提示
- 丰富的组件，如按钮、输入框、表格等
- 支持可动态改变的主题和颜色
- api 设计与原生es一致，易于上手
- 支持按需引入
